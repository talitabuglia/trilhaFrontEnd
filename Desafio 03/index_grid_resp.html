Grid Layout

a) Explique o que é o Grid Layout

Grid Layout é uma tecnologia do CSS que permite configurar o layout em duas dimensões (linhas e colunas), essa junção forma uma grade, o que dá nome a esse sistema.

b) Cite e dê exemplos das propriedades do Grid Container?

Grid container é a tag que envolve os itens do Grid, ao indicar display: grid, essa tag passa a ser um Grid Container.

  • display - Define o elemento como um grid container.
  • grid-template-columns - Define quantas colunas no total serão criadas no grid.
  • grid-template-rows - Define quantas linhas no total serão criadad no grid.
  • grid-template-areas - Define áreas específicas no grid.
  • grid-template - Atalho para definir o grid-template-columns, grid-template-rows e grid-template-areas.
  • gap - Define a distancia entre os elementos do grid.
  • grid-auto-columns - Define o tamanho das colunas do grid implícito (gerado automaticamente, quando algum elemento é posicionado em uma coluna que não foi definida).
  • grid-auto-rows - Define o tamanho das linhas do grid implícito (gerado automaticamente, quando algum elemento é posicionado em uma linha que não foi definida).
  • grid-auto-flow - Define o fluxo dos itens no grid. Se eles vão automaticamente gerar novas linhas ou colunas.
  • grid - Atalho geral para definir qual o tipo de grid.
  • justify-content - Justifica os itens do grid em relação ao eixo x (horizontal).
  • align-content - Alinha os itens do grid em relação ao eixo y (vertical).
  • justify-items - Justifica o conteúdo dos itens do grid em relação ao eixo x (horizontal). Justifica em relação a célula.
  • align-items - Alinha o conteúdo dos itens do grid em relação ao eixo y (vertical). Alinha em relação a célula.

c) Cite e dê exemplos das propriedades do Grid Item?

Grid Item são os filhos do Grid Container. Pode ser explicito ou implícito, explicito é quando você define ele explicitamente no container e implícito é quando ele é criado automaticamente para preencher o conteúdo no grid.

  • grid-column - Define quais colunas serão ocupadas pelo grid item. É possível definir uma linha de início e final, assim o item irá ocupar múltiplas colunas.
  • grid-row - Define quais linhas serão ocupadas pelo grid item.
  • grid-area - Define a área do item do grid. É um atalho para grid-row-start, grid-column-start, grid-row-end, grid-column-end.
  • justify-self - Justifica o item do grid em relação ao eixo x (horizontal). Justifica em relação a célula.
  • align-self - Justifica o item do grid em relação ao eixo y (vertical). Alinha em relação a célula.

d) O que é a unidade fr?

É uma unidade fracional, instrui a uma área a ocupar alguma fração do espaço total disponível.

e) O que é a notação repeat?

Se sua definição contém partes repetidas, você pode usar a notação repeat() para simplificar as coisas.
Ex. grid-template-columns: 20px [col-start] 20px [col-start] 20px [col-start];
Usando repeat(): grid-template-columns: repeat(3, 20px [col-start]);

f) O que são as linhas do Grid?

São as linhas que formam o grid. Diferentes das rows, que traduzindo também significam linhas, porém são propriedades distintas. Por exemplo, uma row possuí duas linhas, uma no inicio e uma no final.

g) O que é uma célula no Grid?

É a menor unidade em um grid, como se fosse uma célula de tabela, uma vez que o grid é definido como pai os itens filhos serão organizados cada um em uma célula do grid definido.

h) O que são áreas do Grid?

Uma ou mais células que compõem a área retangular do grid.
